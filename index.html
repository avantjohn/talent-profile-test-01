Olivia Gray Design System

A modern design system featuring a sophisticated aesthetic with subtle contrasts, cool tones, and thoughtful spacing. This showcase demonstrates the components, colors, and styles defined in Olivia's design profile.

Color System

The color system is based on OKLCH color space with a base color of muted purple (oklch(45% 0.09 270)). The system generates semantic colors with mathematical relationships for tints, shades, and harmony shifts.

Typography

The typography system uses Roboto Flex for headings and body text, with JetBrains Mono for code. It features a type scale ratio of 1.33, with weights ranging from regular (400) to extra bold (780).
